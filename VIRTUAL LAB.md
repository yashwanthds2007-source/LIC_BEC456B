# 2. Introduction

An operational amplifier (Op-Amp) is a direct coupled high gain amplifier consisting of:

- Differential input stage  
- Intermediate stage  
- Level translator  
- Output stage  

The output stage is usually a push-pull complementary symmetry amplifier.

Op-amps are used for:

- DC and AC signal amplification  
- Active filters  
- Oscillators  
- Comparators  
- Regulators

# 4. Ideal Op-Amp Characteristics

- Infinite voltage gain  
- Infinite input resistance  
- Zero output resistance  
- Zero output voltage when input is zero  
- Infinite bandwidth  
- Infinite CMRR  
- Infinite slew rate
